# 📦 Stock-Sense

**Stock-Sense** is a simple Inventory Management System developed as a second-semester university project. Built using **C++** and **Windows Forms**, the project demonstrates core inventory operations with an interactive UI and built-in reporting tools.

## 🚀 Features

- 🔐 **Login System**  
  Basic authentication for user access.

- 📋 **Inventory Management**  
  - Add new products  
  - Categorize items  
  - Edit/update product details  
  - Delete items

- 🔍 **Search Functionality**  
  Quickly search through inventory by name, ID, or category.

- 📈 **Graphical Analysis**  
  Visual graphs to help analyze inventory statistics and trends.

- 📝 **Report Generation**  
  Generate text-based inventory reports.

## 🛠 Tech Stack

- **Language:** C++  
- **Framework:** WinForms (C++/CLI)

## 💻 How to Run

> ⚠️ You must have **Visual Studio** with **C++/CLI** and **.NET Desktop Development** installed.

1. Clone this repository:
   ```bash
   git clone https://github.com/talhaFASTCr123/Stock-Sense.git
   ```

2. Open the `.sln` file in **Visual Studio**.

3. Build and run the project.

> ✅ Alternatively, a precompiled `.exe` file will be available under the **Releases** section for direct execution.

## 📂 File Structure

| File/Folder        | Description                        |
|--------------------|------------------------------------|
| `CompanyFile.h`    | Company-level data structure       |

| `Users.h`          | User credentials and roles         |
| `Reviews.h`        | Customer or internal feedback data |
| `Source.cpp`       | Entry point of the program         |
